# v1.0 — Linux port & GUI configurator

- **Status:** open
- **Target:** v1.0
- **Updated:** 2026-06-30

## Goal
Cross-platform 1.0: Linux support and a graphical config tool.

## Tasks
- [ ] Linux key injection: `src/injector/linux.rs` using `/dev/uinput` (the `uinput` crate),
      behind `#[cfg(target_os = "linux")]` — same `KeyInjector` trait, zero changes elsewhere.
- [ ] udev rule granting non-root access to the G13 (replaces Zadig on Linux).
- [ ] ARM cross-compilation.
- [ ] GUI configurator for editing `config.toml` / profiles.

## Acceptance
Driver runs on Linux with the same config; GUI edits bindings and the running driver
hot-reloads them.

## Notes
- The trait-based injector layer exists precisely to make this a struct swap, not a rewrite.
- Confirm the architecture stayed clean — no Win32 types leaked outside `src/injector/`.
